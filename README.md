# 慧眼探盲导盲智能终端 · 小程序

## 📱 项目概述
本项目是为“慧眼盲探——基于计算机视觉的智能导盲终端”配套的**亲友端微信小程序**。旨在搭建视障人士与家属之间的信息桥梁，让家人实时了解使用者的出行状态，并在紧急情况下提供及时帮助。

---

## 🎯 核心功能（仅实现部分）

### 1. 实时定位追踪
- 地图实时显示使用者当前位置
- 记录每日出行轨迹与常走路线
- 支持历史轨迹回放
<img width="559" height="1097" alt="image" src="https://github.com/user-attachments/assets/fa3a12a1-47d8-45e2-a3b2-6877118026fe" />

### 2. 设备状态监控
- 导盲终端电量实时显示（低电量自动提醒）
- 设备在线/离线状态监测
- 设备异常告警（如跌落、碰撞）

### 3. 紧急求助联动
- **一键求助**：使用者按下SOS按钮，亲友端立即接收告警推送
- **实时通话**：支持亲友与使用者双向语音联络
- 自动推送求助位置与现场环境描述
<img width="1926" height="1599" alt="image" src="https://github.com/user-attachments/assets/62c44731-1e4d-4cff-a705-a66e024589ff" />


### 4. 出行规划与记录
- 行程统计：每日/每周出行次数、时长、里程
- 常用路线保存与快速导航
- 出行安全报告（可分享给其他家庭成员）

### 5. 亲友群组管理
- 支持多位家属同时绑定同一设备
- 群组内消息互通（如“已安全到家”）
- 权限分级：管理员（可解绑设备）、普通成员（仅查看）

## 🛠️ 技术栈

| 模块 | 技术选型 |
|------|----------|
| 前端框架 | 微信小程序原生 + WXML + WXSS |
| 地图服务 | 腾讯地图/高德地图 SDK |
| 实时通讯 | WebSocket / 微信小程序即时通讯能力 |
| 数据存储 | 微信云开发 / 腾讯云数据库 |
| 设备对接 | 通过后端API与导盲终端通信（HTTP/MQTT） |
